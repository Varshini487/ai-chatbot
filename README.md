# 💬 AI Chatbot

A **Conversational AI Chatbot** with memory, context management, and multi-turn dialogue.

## ✨ Features
- 🧠 **Conversation memory** — remembers context across turns
- 🌐 **Multi-language support**
- 🔧 **Custom personas** via system prompts
- 📁 **Knowledge base integration** (optional RAG)
- ⚡ **Streaming responses**

## 🛠️ Tech Stack
- **OpenAI GPT-4 / Gemini** – language model
- **LangChain** – memory and chain management
- **Redis / SQLite** – conversation persistence
- **FastAPI + WebSockets** – real-time backend
- **Streamlit** – chat interface

## 🚀 Getting Started
```bash
git clone https://github.com/Varshini487/ai-chatbot
cd ai-chatbot
pip install -r requirements.txt
streamlit run app.py
```

- Customer support automation
- Educational tutoring assistant
